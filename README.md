# Аguteens
Проект представляет собой главную страницу для образовательного проекта Aguteens. Это современный веб-сайт с использованием современных технологий разработки.

## Описание проекта
Это адаптивный веб-сайт с интерактивными компонентами, созданный для предоставления информации об образовательных программах и услугах. Проект использует современные подходы к разработке с акцентом на переиспользуемые компоненты и чистый код.

### Технологии
Основные технологии:

TypeScript - для типобезопасной разработки

Webpack - модульный сборщик

Less - препроцессор стилей

ESLint - линтер JavaScript/TypeScript

Stylelint - линтер CSS/LESS

### Архитектура
Компонентный подход к разработке

Модульная структура проекта

Поддержка Typescript типов

Автоматическая проверка качества кода

Структура проекта:

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── modal/           # Модальные окна
│   ├── list/            # Списки с раскрывающимися элементами
│   ├── horizontal-scroll/ # Горизонтальная прокрутка
│   ├── spoiler/         # Спойлеры
│   ├── replace-image/   # Замена изображений
│   └── ...              # Другие компоненты
├── common/             # Общие стили и файлы
└── index.ts            # Точка входа проекта
```
### Возможности
Интерактивные компоненты:

Модальные окна - с возможностью переключения между контентом

Списки с раскрывающимися элементами - FAQ и другие списки

Горизонтальная прокрутка - бесконечная прокрутка карточек

Спойлеры - анимированные раскрывающиеся блоки

Замена изображений - динамическая смена фоновых изображений

## Инструменты разработки:
Линтинг - автоматическая проверка качества кода

Сборка - оптимизированная сборка для продакшена

## Компоненты проекта

Описание компонентов

### Modal (Модальное окно)
Компонент для создания модальных окон с возможностью переключения между несколькими контентными элементами.

Атрибуты:

`data-modal-bg-color` - цвет фона модального окна

`data-modal-content` - CSS селектор контента модального окна
Пример использования:

```html
<!-- Кнопка открытия модального окна -->
<div data-modal-bg-color="#fff" data-modal-content="#modal-content">Открыть</div>

<!-- Контент модального окна -->
<div id="modal-content" style="display: none;">
  <p>Содержимое модального окна</p>
</div>
```

```typescript
// Инициализация модальных окон
initModalListeners();
```

### List (Список с раскрывающимися элементами)
Компонент для создания списка с возможностью раскрытия/скрытия элементов.

Пример использования:


```HTML
<div class="list">
  <div class="list__item">
    <div class="list__question" data-index="0">Вопрос 1</div>
    <div class="list__answer" data-index="0">
      <div class="list__answer__content">Ответ 1</div>
    </div>
  </div>
</div>
```
```typescript
// Инициализация списков
initList(element, openedIndex);
```

### HorizontalScroll (Горизонтальная прокрутка)
Компонент для создания горизонтальной прокрутки с бесконечным циклом.

Параметры:

`direction` - направление прокрутки ('left' или 'right')

`infinite` - флаг бесконечной прокрутки

`controls` - показывать ли элементы управления

Пример использования:


```HTML
<div class="block-cases__cards">
  <!-- Содержимое карточек -->
</div>
```

```typescript
// Инициализация горизонтальной прокрутки
initInfiniteScroll(element, { direction, infinite, controls });
```

### Spoiler (Спойлер)
Компонент для создания спойлеров с анимированным раскрытием.

Пример использования:


```HTML
<div class="spoiler">
  <div class="spoiler__title">Заголовок спойлера</div>
  <div class="spoiler__content">
    <p>Содержимое спойлера</p>
  </div>
</div>
```
```typescript
// Инициализация спойлеров
initSpoiler(document.querySelector('.spoiler'));
```

### ReplaceImage (Замена изображения)
Компонент для замены фонового изображения по клику.

Атрибуты:

`data-replace-bg-image-src` - ссылка на новое изображение

`data-replace-bg-target` - CSS селектор элемента для замены фона
Пример использования:


```HTML

<div data-replace-bg-image-src="/new-image.jpg" data-replace-bg-target=".target-element">
  Клик для замены
</div>
```

## Для заказчиков: установка и развертывание проекта aguteens-main-page на Windows с Node.js

Необходимые требования

Windows 10 или выше

Node.js версии 18 или выше (рекомендуется LTS версия)

Git (для клонирования репозитория)

### Шаг 1: Установка Node.js
Скачайте Node.js с официального сайта: https://nodejs.org/

Выберите LTS (Long Term Support) версию для стабильной работы

Запустите установщик и следуйте инструкциям

После установки проверьте версию:
```bash
node --version
npm --version
```

### Шаг 2: Установка Git
Скачайте Git с официального сайта: https://git-scm.com/downloads

Запустите установщик и следуйте инструкциям

Проверьте установку:
```bash
git --version
```

### Шаг 3: Клонирование репозитория

Откройте командную строку или PowerShell

Перейдите в директорию, где хотите разместить проект:

```bash
cd C:\projects
```
Клонируйте репозиторий:
```bash
git clone https://github.com/avel2k6/aguteens-main-page.git
```
### Шаг 4: Установка зависимостей
Перейдите в папку проекта:
```bash
cd aguteens-main-page
```
Установите все необходимые зависимости проекта:
```bash
npm install
```
### Шаг 5: Проверка установки
Проверьте, что все зависимости установлены корректно:
```bash
npm run lint
```

Проверьте сборку проекта:

```bash
npm run build
```
Если сборка прошла успешно, вы увидите сообщение об успешной компиляции в директории

### Шаг 6: Запуск в режиме разработки
Для запуска сервера разработки выполните:
```bash
npm run dev
```

После запуска сервера откройте браузер и перейдите по адресу http://localhost:8080

### Шаг 7: Сборка для продакшена
Для создания production-сборки выполните:
```bash
npm run build
```
Скомпилированные файлы будут находиться в директории docs/
